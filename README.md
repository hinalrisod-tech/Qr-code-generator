# Qr-code-generator
A simple QR code Generator built using python that converts text or URLs into qr code.
##features
-generate QR code from text or URL
Save QR code as image
easy to use

##Technologies used
-python
-qrcode library

##How to Run
install required library.
pip install qrcode(pil)

## use case 
you can generate QR codes for lonks, or any text data.
